## tmux-config

## Prefix Key
 CTRL

## Sessions
- List Sessions - s
- Switch to last Session - S
- Attach Session w/ name completion - g
- Create Session - C
- Detach Session - d
- Rename Sessions - $
- Current pane as new Session - @

## Windows
- List Windows - w
- Create Window - c
- Rename Window - ,
- Next Window - n
- Previous Window - p
- Windows 0 - 9
- Swap Windows - < >

## Panes
- Show Panes #'s - q
- New Vertical Pane \
- New Horizontal Pane -
- Toggle Panes - o
- Navigate Panes - (VIM arrow keys) HJKL
- Cycle Pane Locations - Ctrl-o
- Zoom Pane - z

## Misc
- Reload tmux source - R
- Resurrect save - Ctrl+s
- Resurrect restore - Ctrl+r
